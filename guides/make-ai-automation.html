אנשים מדברים על אוטומציה כאילו זה מדע טילים. זה לא.

אוטומציה אמיתית - שכל אחד יכול לבנות - היא להגיד לכלי אחד "כשקורה X, תעביר ל-Y". אם תוסיף AI באמצע - יש לך מערכת חכמה שמטפלת בעצמה במשימות שיוצאות לך זמן.

במדריך הזה אתה הולך לבנות מערכת אמיתית, מתחילה לסוף, שעובדת לבד 24/7. הדוגמה: כשמישהו ממלא טופס באתר שלך, Claude יסכם, ו-Notion יקבל את המידע - ויישלח לך מייל סיכום. 30 דקות. חינם.

1. למה Make ולא Zapier?

שתיהן עושות אותו דבר - מחברות בין שירותים. ההבדל:

  • Zapier - יותר פופולרי, ממשק קצת יותר חברותי, מחיר גבוה יותר
  • Make.com - יותר חזק, גרפי, חינמי עד 1,000 פעולות בחודש (פי 10 מ-Zapier)

למתחילים - שתיהן עובדות. למי שרוצה לגדול - Make. במדריך הזה נשתמש ב-Make.

2. הדוגמה שתבנה

תארו לעצמכם: יש לכם דף נחיתה לעסק. כשמישהו ממלא טופס פניה - אתם רוצים שהמידע יגיע למקום אחד מסודר, עם סיכום חכם של מה שהאדם כתב, כדי שתוכלו להחליט מהר אם להחזיר אליו.

זה התרשים:

1

טופס יצירת קשר ממולא

מבקר באתר ממלא שמו, הודעה, ומטרת הפנייה

2

Make מקבל את הנתונים

webhook שמקבל ת התוכן ומעביר לסוכן AI

3

Claude מסווג ומנסח

מזהה אם זה ליד חם/קר, וכותב תגובה מותאמת בעברית

4

שולח מייל + נוטיפיקציה

תגובה ללקוח, סיכום למייל שלך, רישום ב-Notion

טופס נשלח → Claude קורא ומסכם בעברית → שורה חדשה ב-Notion עם הסיכום → מייל אליי עם הפרטים והסיכום.

למה זה משנה: במקום לקרוא 5 דקות כל מייל פניה ולהחליט אם זה רציני - אתה מקבל מייל בן 2 שורות עם סיכום של מה שהאדם רוצה. הופך טיפול בלידים ממשימה למקבץ של החלטות בני 10 שניות.

3. הצטיידות - 5 דקות

תפתחו 4 חשבונות (כל אחד חינם):

  1. make.com - הרשמה רגילה. גרסה חינמית מספיקה.
  2. notion.so - אם אין כבר. צרו workspace חדש.
  3. anthropic.com - חשבון API ל-Claude. קבלו $5 חינם להתחיל.
  4. Gmail - יש לרובכם.

בנוסף תצטרכו טופס. אם יש לכם אתר עם טופס - מצוין. אם אין - תשתמשו ב-tally.so (חינם, 30 שניות הקמה).

4. שלב 1: בניית הסצנריו ב-Make

נכנסים ל-Make → Create new scenario. רואים מסך ריק עם עיגול אחד גדול באמצע - זאת התחנה הראשונה.

  1. לוחצים על העיגול → מחפשים "Webhooks" → בוחרים "Custom Webhook"
  2. Add → נותנים לו שם ("Form submissions") → OK
  3. Make ייצר עכשיו URL ייחודי. תעתיקו אותו. הוא נראה ככה: https://hook.eu2.make.com/abc123xyz

זה ה-URL שהטופס שלך יישלח אליו.

5. שלב 2: חיבור הטופס

אם אתה משתמש ב-Tally:

  1. פותחים את הטופס בטאלי → Integrations → Webhooks
  2. מדביקים את ה-URL שהעתקתם
  3. שולחים בדיקה - ממלאים את הטופס פעם אחת

אם יש לך אתר עם טופס משלך (כמו /api/subscribe שיש ב-yarinmalka.co.il) - תוסיף בקוד שורה שתשלח גם ל-webhook URL הזה.

הסבר: webhook זה כמו "טריגר" - כתובת שכל פעם שמישהו שולח אליה מידע, היא מפעילה את האוטומציה שלך.

6. שלב 3: Claude מסכם

חוזרים ל-Make. עכשיו ל-Webhook יש כבר נתונים מהבדיקה. נחבר אותו ל-Claude:

  1. + ליד עיגול ה-Webhook → מחפשים "Anthropic" → בוחרים "Create a Message"
  2. נותנים את ה-API Key מ-anthropic.com (Settings → API Keys)
  3. בוחרים מודל: claude-haiku-4-5 (מספיק טוב, הכי זול)
  4. בשדה "Messages" כותבים פרומפט: תכין סיכום קצר של 2 שורות בעברית של הפניה הבאה: שם: {name} | טלפון: {phone} | הודעה: {message}
  5. במקום {name}, {phone}, {message} - תבחרו את השדות מה-Webhook (Make מציע אותם בתפריט)
  6. Max tokens: 200 (סיכום קצר)

7. שלב 4: שמירה ב-Notion

קודם תכינו ב-Notion טבלה (database) חדש בשם "Leads" עם העמודות:

  • Name (Title)
  • Phone (Phone)
  • Email (Email)
  • Original Message (Text)
  • AI Summary (Text)
  • Date (Created time - אוטומטי)
  • Status (Select: New / Contacted / Closed)

חוזרים ל-Make:

  1. + אחרי Claude → "Notion" → "Create a Database Item"
  2. מתחברים ל-Notion (אישור חד-פעמי)
  3. בוחרים את הטבלה "Leads"
  4. לכל עמודה - בוחרים את השדה המתאים מה-Webhook או מ-Claude (השדה Summary מ-Claude)

8. שלב 5: מייל אליך

  1. + אחרי Notion → "Gmail" → "Send an Email"
  2. To: האימייל שלך
  3. Subject: פניה חדשה: {name}
  4. Body: שם: {name}
    טלפון: {phone}

    סיכום AI:
    {summary}

    הודעה מקורית:
    {message}

9. בדיקה

לוחצים Run once ב-Make. ממלאים את הטופס פעם אחת. תוך 5 שניות:

  • שורה חדשה ב-Notion עם הסיכום
  • מייל אצלך
  • במסך ה-Make - כל השלבים מסומנים בירוק

אם הכל עובד - תפעיל את הסצנריו (טוגל למעלה משמאל מ-Off ל-On). מהרגע הזה הוא רץ אוטומטית.

ברכות. בנית מערכת AI אמיתית. כל פעם שמישהו ימלא את הטופס שלך, היא תרוץ לבד.

10. איך להרחיב מכאן

הבסיס הזה נותן לך הרבה יותר ממה שזה נראה. רעיונות הרחבה ב-30 דקות נוספות:

  • סיווג לידים אוטומטי - תוסיף שלב Claude נוסף שמחזיר "Hot/Warm/Cold" לפי טון ההודעה. ב-Notion יוצא Status אוטומטי.
  • WhatsApp במקום מייל - תחליף את ה-Gmail ב-Twilio (יש מודולים מוכנים)
  • תיוג חכם - Claude יחזיר 3 תגיות (פיתוח/פיננסים/סדנה) שיתויגו אוטומטית בנוטיון
  • תזכורות אוטומטיות - אם אחרי 3 ימים ה-Status עוד "New", הסצנריו ישלח לך תזכורת

11. שגיאות נפוצות

"Webhook not receiving data" - תבדקו ש-Tally/הטופס שולח באמת. רוב הזמן זה הגדרה לא נכונה של ה-URL.

"Anthropic API error 401" - ה-API Key שגוי או שלא טענת קרדיט בחשבון. תכניסו $5 ראשוניים ב-anthropic.com → Billing.

"Notion permissions" - תוודאו שהטבלה ב-Notion שיתפת עם integration של Make (פותחים את הטבלה → Share → Add Make integration).

"Sent twice" - לפעמים הטופס שולח 2 פעמים. תוסיפו ב-Make מסנן (Filter) שמתעלם משליחות חוזרות תוך 60 שניות מאותו טלפון.
