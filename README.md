# yarinmalka.co.il

Personal site for Yarin Malka — AI · TECH · TOOLS.

Static HTML/CSS, deployed via Cloudflare Pages.

## Structure

- `index.html` — homepage
- `guides.html` — guide index
- `guides/` — individual guide pages
- `about.html` — about page
- `signup.html` — newsletter signup
- `assets/` — CSS, images, screenshots

## Deploy

Push to `main` → Cloudflare Pages auto-deploys to `yarinmalka.co.il`.
